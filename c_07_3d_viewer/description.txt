3DViewer_v1.0
О программе Программа для просмотра 3D моделей в каркасном виде с реализацией афинных преобразований и настройки отображения. Настройки сохраняются между запусками программы.

    Перемещение в трехмерном пространстве
    Поворот в трехмерном пространстве
    Масштабирование

Настройки

    Смена типа проекции (параллельная и центральная)
    Настройка типа ребер (сплошной, пунктирный)
    Настройка цвета ребер
    Настройка толщины ребер (1-10)
    Настройка способа отображения вершин (круг, квадрат, отключено)
    Настройка цвета вершин
    Настройка размера вершин (1-10)
    Настройка цвета фона

Запись

    Возможность сохранения отрендеренных изображений в формате .bmp и .jpeg.
    Возможность записи скринкастов в gif-анимацию.

Установка Для установки программы введите команду make install P.S. Установку и запуск приложения осуществлять через отдельный терминал, через встроенный терминал VS Code бывают баги. 

Удаление Для удаления программы используйте команду ``make uninstall``.
